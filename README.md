# 👕 Vibe Thrift — Online Store Demo

This is a clean, mobile-friendly landing page built for a vintage clothing store in Nairobi, Kenya. 

## 🔗 Live Website
* **Click here to see the live project:** (https://vibethrift.netlify.app/)

## 💡 What this project does
* **Mobile First:** Designed to look perfect on smartphones, which is how most people shop in Kenya.
* **Easy Ordering:** Customers can look at unique clothing items and click a button to order them instantly via WhatsApp.
* **Local Features:** Includes clear sections for Lipa na M-Pesa, G4S delivery, and buyer reviews to build customer trust.

## 🛠️ Code Tools Used
* **HTML5 & CSS3:** For the layout, fonts, colors, and responsive design.
* **JavaScript:** Used to handle the interactive buttons and the WhatsApp chat links.
